# Fonts Directory

This directory should contain Noto Sans fonts for Hindi (Devanagari) support.

## Required Fonts:

1. **NotoSans-Regular.ttf** - Regular weight Noto Sans
2. **NotoSans-Bold.ttf** - Bold weight Noto Sans  
3. **NotoSansDevanagari-Regular.ttf** - Regular weight Noto Sans Devanagari
4. **NotoSansDevanagari-Bold.ttf** - Bold weight Noto Sans Devanagari

## Download Links:

- Noto Sans: https://fonts.google.com/noto/specimen/Noto+Sans
- Noto Sans Devanagari: https://fonts.google.com/noto/specimen/Noto+Sans+Devanagari

## Quick Download (using curl or browser):

1. Visit Google Fonts
2. Download Noto Sans family
3. Download Noto Sans Devanagari family
4. Extract and place the .ttf files in this directory

## Alternative: Use system fonts

If you don't want to bundle fonts, you can modify pubspec.yaml to use system fonts instead.
